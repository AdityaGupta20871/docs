# Spheron Public IPFS Gateway

Spheron Public Gateway is our public node that you can use to access content on IPFS

Gateway: `https://ipfs.spheron.link`

An example of accessing data: https://ipfs.spheron.link/ipfs/bafybeiedojjozm3qyhd6zu6q7dq3e6vhy3a7sfr5m6w34fpzyrxmq2xocq/

<Callout type="warning">
  The Spheron Public Gateway is designed for testing purposes and should not be
  used on production. Check out the [Dedicated Gateways](/gateways/dedicated) for
  production usage.
</Callout>
